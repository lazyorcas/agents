# Theming

Find and use `BRANDING.md` for branding guidelines.

## Boilerplate

1. Open [shadcn's Create Project](https://ui.shadcn.com/create) in the browser
2. Ask the author to select the fields
3. Click on "Create Project"
4. Ask the author to pick the template
4. Copy the `npx` command and run it in the Terminal

## Filling the gap

For the branding elements that are not covered by the boilerplate theme generated by shadcn, make sure they are covered in custom implementations.
